Este tema vem com a integração [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) instalada e configurada no seu arquivo `astro.config.mjs`. Se você preferir não usar MDX, pode desabilitar o suporte removendo a integração do seu arquivo de configuração.

## Por que MDX?

MDX é um tipo especial de Markdown que suporta sintaxe JavaScript e JSX incorporada. Isso permite [misturar JavaScript e Componentes de UI no seu conteúdo Markdown](https://docs.astro.build/en/guides/markdown-content/#mdx-features) para coisas como gráficos interativos ou alertas.

Se você tem conteúdo existente escrito em MDX, esta integração deve facilitar a migração para o Astro.

## Exemplo

Aqui está como você pode usar recursos MDX nas suas postagens do blog. MDX permite usar componentes diretamente no seu markdown.

**Exemplo de Botão Interativo:**

<button onclick="alert('Olá do MDX!')">
	Clique em mim - Componente incorporado no MDX
</button>
